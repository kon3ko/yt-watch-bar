# Privacy Policy YT Watch Bar does not collect, store, transmit, or share any personal information or user data. This extension only modifies the appearance of YouTube's watched progress bar to improve visibility and allow custom colors selected by the user. All settings are stored locally within the user's browser using Chrome storage APIs. No data is sent to external servers. The extension does not: * Collect personal information * Track browsing activity * Use analytics or telemetry * Share data with third parties * Require user accounts or login If you have any questions regarding this privacy policy, please contact the developer through the extension page.
